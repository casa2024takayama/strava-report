## 1. 月間総評 — 量・質・バランスの評価

**総距離 41.6 km（7アクティビティ）、実質2日間の練習**

まず最初に確認しておきたいのは、**すべての練習が6月5日（金）〜7日（日）のわずか3日間に集中している**点です。月間を通じて走れていたわけではなく、週1回程度しか練習できていない状況と読み取れます。それを踏まえた上で評価します。

| 項目 | 評価 | コメント |
|------|------|---------|
| 月間距離 | △ | 41.6 km は月間距離として少なめ。マラソン完走を目指すなら最低でも月150〜200 km が目標ライン |

## 2. 週別評価

### 第1週（6月1日〜7日）：41.6 km

データ上は1週間にすべての練習が収まっていますが、実際の練習日は金・土・日の3日間です。

**良かった点**
- 金曜の Night Run でしっかりとしたE走ベースを積んだ後、土曜・日曜で高強度走を実施するという「積み上げ→刺激」の流れは、意図せずともダニエルズ的な構造に近い
- 高強度走（3:40〜3:55/km 台）の後に、クールダウン目的のE走を実施しており、セッション設計の意識が見える

**課題**

- 月の大半（1〜4日、8〜30日）が空白。**継続性の欠如が最大の課題**
- 週間距離 41.6 km を2〜3日でこなすのと、5〜6日に分散させるのでは、体への適応効果がまったく異なる

## 4. ペース・心拍分析 — VDOT 推定とゾーン配分

### VDOT 推定

高強度走（Evening Run 5.0 km｜3:48/km）から推定します。

- 5 km を 3:48/km（= 19分00秒）で走れる能力があるとすると、**VDOT ≒ 52〜54** と推定されます
- これはフルマラソン換算で約 **3時間30〜37分** 相当の走力です

ただし、**この走力を安定して発揮するための有酸素基盤（E走の積み重ね）が現状では不足しています。**
