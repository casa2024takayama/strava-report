## 1. 月間総評

**月間距離：176.7 km / 18アクティビティ / 平均HR 141 bpm**

まず、6月も高い継続性でトレーニングを積めたことを評価します。月間176kmは、マラソンを本気で取り組むランナーとして十分な練習量です。週間距離が40〜53kmの範囲で推移し、大きな断絶がなかった点も好材料です。

一方で、全体を通じて見えてくる最大の課題が「**1日に複数のアクティビティを重ねる形式**」です。6月6日（3本）、6月7日（3本）、6月14日（2本）、6月24日（2本）、6月27日（2本）と、同日に分割してセッションを行うケースが非常に多く見られました。ガーミンなどのデバイスで記録が分かれているだけの場合もありますが、データを見る限り、**ウォームアップ・クールダウンを別アクティビティとして記録しているセッション**と、**本質的に別目的で走った2本**が混在しています。

特に問題なのが「**同日にEasyランとR（レペ）強度のランを組み合わせる**」パターンです。6月6日・7日・14日にその傾向が顕著で、ダニエルズメソッドの「ハード翌日はイージーまたは休養」という根本原則を圧迫しています。

**月間総評：B−**
量は合格点、しかし強度管理と回復の設計に課題あり。

## 2. 週別評価

### 第1週（6/1〜6/7）：41.6 km ⚠️

この週は最も気になる構成です。

- 6/5（金）Night Run 10.7km：後半に強度が入ったテンポ気味のラン
- 6/6（土）：**3本合計17.3km**。うち1本（Evening Run 5km）は3:48/kmという全力に近いRペース。平均HR 181、最大HR 192という数値は、最大心拍数付近での走行を示しています
- 6/7（日）：**3本合計13.7km**。うち1本（3.1km）は3:52/kmでHR 178と高強度

金→土→日と3日連続で強度のある練習が入っており、ダニエルズの原則「クオリティセッションは週2回まで・ハード後は必ずイージー」を大幅に逸脱しています。疲労が抜ける前に次の強度練習を重ねてしまうと、VO2max向上どころか慢性疲労のリスクを招きます。

**週評価：C**
練習の熱意は伝わりますが、構成上は要修正。

## 3. 個別アクティビティコメント

### ✅ 6/19（金）夜のランニング 14.3km — **今月のベストEasyラン**

全15ラップが5:41〜6:22/km、平均HR 128という教科書通りのEasyペース。ロング走（6/21）の2日前にしっかり回復走を入れられており、コンディション作りとして非常に優秀です。このペースと心拍数の感覚を「Easyの基準」として今後も維持してください。

---

### ✅ 6/21（日）朝のランニング 33.3km — **圧巻のロング走、ただし後半の扱いに注意**

33.3kmという距離はフルマラソン準備として意欲的な取り組みです。前半（1〜14km）は5:06〜5:32/kmでHR 125〜148と安定した入りで、15〜21km付近で4:35〜4:58/kmにペースアップ、HR 160〜179まで上昇しています。

**良い点：**
- 前半20kmはほぼEasy〜Mペース範囲で安定
- 後半も足が動いており、体力的な底上げを感じます

**改善点：**
- Lap15〜21でのペースアップ（4:35/km、HR 161〜179）はロング走中にしてはやり過ぎです。ロング走の目的は有酸素基礎の構築であり、ダニエルズはロング走のペースを一貫してEペース（会話ができる強度）に留めるよう強調しています
- 特にLap17の4:35/km（HR 161）、Lap20の4:38/km（HR 179）はTゾーン〜Iゾーン境界に達しており、これはロング走の目的外の強度です
- 結果的に後半（Lap22〜34）でペースが大幅に落ち（5:24〜6:17/km）、HRも高止まりしています。これは中盤で出力を使いすぎた典型的なパターンです

**提言：** ロング走は最後まで「もう少し走れる」と感じる強度を保つことが原則です。中盤でのペースアップ欲求をグッとこらえ、終始5:30〜6:00/km（現状のEasyレンジ）に徹してください。距離は33kmで十分すぎるほどですが、一定ペースで走り切ることで適応効果が高まります。

## 4. ペース・心拍分析

### VDOT推定

現時点でのレースデータがないため、練習ペースと心拍数から推定します。

- 6/19のEasyラン：平均5:59/km・HR 128
- 6/21のロング走前半：平均5:15/km・HR 140前後
- 高強度区間：3:50〜4:00/km・HR 185前後

これらの関係から、**推定VDOT：50〜53**程度と見られます。マラソン換算で概ね**3:30〜3:45**のポテンシャルがあると考えられます。ただし、VDOT確定のためにはハーフマラソンや10kmのレース結果が最も正確です。

### ゾーン配分の評価

| ゾーン | 推定ペース目安（VDOT51想定） | 6月の実態 |
|--------|----------------------------|----------|
| E（Easy） | 6:00〜6:30/km | 主要な練習の多くはここ。◎ |
| M（Marathon） | 5:10〜5:20/km | 一部ロング走中盤。ただし意図が不明確な場面も |
| T（Threshold） | 4:45〜4:55/km | 複数のランで散発的に登場。系統的でない |
| I（Interval） | 4:05〜4:15/km | 6/21後半・6/27 Lap7など。意図せず混入気味 |
| R（Repetition） | 3:45〜4:00/km | 6/6・6/7の短距離高強度セッション |

**問題点：** TゾーンとIゾーンの練習が「意図的なクオリティセッション」として独立しておらず、Easyランやロング走の途中にスパートとして混入しています。これでは各ゾーンへの適切な刺激と回復が設計できず、「何となく疲れるが向上している実感が薄い」状態になりやすいです。

ダニエルズが推奨するのは「**Easy/Easyの日はとことんEasy、クオリティの日は計画的なT走またはインターバル**」という明確な区分けです。

**心拍数の上限値について：**
最大HR 195（6/27）・192（6/6）・188（6/21他）が複数回記録されています。これがHRmax付近であるとすれば、Easyゾーンの目安はHRmax × 74% = 約140〜144bpm以下です。6月の平均HR 141bpmは月全体ではEasyゾーン内に収まっていますが、個別セッション内での高強度スパイクが頻発している点は看過できません。
