## 1. 月間総評

**月間距離 176.7km・18回は量として十分な水準**ですが、中身を精査すると「量はこなせているが、質の設計が不安定」という印象です。

最大の懸念はGarminの生理指標です。**VO2max が60.0→57.4へ低下・トレーニングステータスがUNPRODUCTIVE・トレーニングレディネスの平均が44/100・安静時心拍の上昇（44→45bpm）**——これらは揃って「疲労蓄積・オーバーリーチの入口」を示しています。練習量自体が多いわけではないのにこの状態になっている背景には、**睡眠時間の短さ（平均6.6時間）と、1日複数アクティビティへの分割による回復の分断**が強く影響していると推察されます。

練習の構成を見ると、イージーランが適切に実施されている一方で、**レペティション（3:48〜4:00/kmクラスの全力走）が毎週のように出現**しており、しかもその多くが「ウォームアップ→全力→クールダウン」をそれぞれ別アクティビティとして記録する形になっています。この分割記録は負荷の全体像を見えにくくするため、コーチング上も自己管理上も注意が必要です。

ダニエルズの原則「週2回のクオリティ上限」に照らすと、6月は毎週クオリティセッションが2〜3回含まれており、そのいずれもが単独のEラン日をきちんと挟まずに連続していることがあります。**量を積みながら強度も削れていない——これがUNPRODUCTIVEステータスを招いた核心**です。

## 3. 個別アクティビティコメント

### ✅ 6/21(日) 朝のランニング 33.3km — 月間ベストの練習

今月最も評価できるアクティビティです。

**前半（Lap1〜14：5:06〜5:32/km、HR124〜148）**は非常に安定したMゾーン寄りのEペースで推移。**中盤（Lap15〜21：4:35〜5:15/km、HR161〜179）**で自然にペースアップし、後半（Lap22〜34：5:24〜6:17/km、HR145〜157）で適切に落ち着いています。

特に評価できるのは、**Lap17〜20で4:35〜4:40/kmというMペース以上の領域を維持しながら、Lap22以降に自力でペースをコントロールして落とせている**点です。後半崩れずにきれいにクールダウンできています。ただし最大HRが184bpmに達しており、中盤の4:35付近はVDOT57レベルではTゾーン上限に近い強度です。意図してテンポ区間を入れたのであれば問題ありませんが、**ロング走中にこの強度まで上げることは疲労蓄積を加速させる**ため、ロング走はE〜Mペース（5:10〜5:40程度）に留めることを推奨します。

### ⚠️ 6/27(土) Lunch Run 21.5km — ペース配分の乱れが目立つ

Lap1〜4（5:22〜5:47）は良いEペース。しかしLap5から突如ペースが上がり始め、**Lap6(4:26)→Lap7(3:54・HR188)**へと急上昇。その後Lap8〜19で5:30〜5:58に戻るものの、終盤Lap20(5:03)→Lap21(4:46)→Lap22(4:47)と再びペースアップしています。

これは**「2段ロケット型」の無計画なペース変動**で、中盤の3:54/km・HR188bpmはほぼ最大強度です。ロング走中に2度の急激なペースアップが入るのはダニエルズのロング走の概念と相容れません。ロング走はE強度で一定ペースを保ち、有酸素基礎を積むことが目的です。競うものがあったのかもしれませんが、意識的に「今日はロング走」と目的を決めてから走るようにしましょう。

同日さらに**Afternoon Run 5.3kmを追加**しているのも要注意です。21.5km走後に5.3kmは回復の観点から望ましくありません。

### ⚠️ 6/6(土) Evening Run(R) 5.0km — 強度は高いが前後の配置が問題

5km・平均HR181・最高192bpmは明らかにRペース（3:43〜4:00/km）での高強度走です。走力そのものを見れば、VDOT57〜60のランナーがRペース3:45前後を走れているのは実力の裏付けになっています。問題は**前後の配置**です。直前に7.3kmのウォームアップ的な走りがあり、直後は何もなく翌日(6/7)に再び高強度走が入っています。Rペース走は高品質なリフレッシュと回復時間（セット間・日間）がセットで初めて効果を発揮します。

### ✅ 6/19(金) 夜のランニング 14.3km — 理想的なEラン

全ラップ5:41〜6:22/km・HR116〜132bpmで揃っており、**完璧なEゾーン走**です。翌々日のロング走(6/21)に向けた準備として申し分ありません。このような「目的を持ったイージーラン」をもっと増やすことが、PRODUCTIVE化への鍵になります。

### ⚠️ 6/7(日) ランチタイムランニング(R) 3.1km — 翌日を見据えた計画性が必要

3km・平均HR178・3:52/km平均の高強度走。前日(6/6)にすでにR強度走を行った翌日に、再び全力に近い走りを入れています。この2日連続の高強度がレディネスを急速に低下させ、週明けの練習の質を下げる一因になっています。

## 4. ペース・心拍分析

### VDOT推定

GarminのVO2max推定が57.4（月初60.0から低下）を示しています。6/21のロング走で4:35〜4:40/km時にHR161〜179bpmに達していることを踏まえると、**現在の実力ベースのVDOTは概ね55〜57の範囲**と推定するのが現実的です。VO2maxの低下は疲労による機能低下であり、トレーニングの「実力低下」ではありませんが、疲弊した状態でVDOTを参照してペース設定することの危険性は理解しておく必要があります。

### 各ゾーンの目安ペース（VDOT56を基準）

| ゾーン | ペース目安 |
|---|---|
| E（イージー） | 5:40〜6:20/km |
| M（マラソンペース） | 5:00〜5:10/km |
| T（テンポ） | 4:35〜4:45/km |
| I（インターバル） | 4:05〜4:15/km |
| R（レペティション） | 3:45〜4:00/km |

### ゾーン配分の評価

今月のアクティビティを心拍・ペースから分類すると、おおよそ以下の通りです。

- **Eゾーン（5:40〜6:30/km・HR120〜145bpm程度）**：全体の約65〜70%
- **MゾーンからTゾーン（5:00〜5:20/km・HR145〜160bpm）**：約15〜20%
- **R相当（3:45〜4:10/km・HR175〜192bpm）**：断続的に出現

**Rペースが週に1〜2回必ず出現している**のが今月の特徴です。ダニエルズの80/20ルールに照らすと、クオリティの絶対量は多くはありませんが、**その質（=Rペース）が高すぎる週が続いている**ことが問題です。R走は走経済性の向上に有効ですが、週1回・400〜200m×数本の制御されたセッションとして行うべきものです。今月のような「ロング走の終盤に全力疾走」「翌日にまた全力走」という形は、身体への負荷計算が難しく回復を見誤りやすいです。

### HRと心拍ドリフトについて

6/21のロング走では後半のHRが150〜157bpmで安定しており、心拍ドリフト（同一ペースで心拍が徐々に上昇する現象）が比較的小さく抑えられています。これは有酸素基礎がしっかり積み上がっている証拠であり、**6月のEランの積み重ねは確実に効いています**。7月も土台となるEランを大切にしてください。

## 5. 2026年7月の練習提案

### 現状認識と方針

月末時点でVO2max低下・UNPRODUCTIVE・レディネス低下・安静時心拍上昇という複合的なオーバーリーチサインが出ています。7月の前半（少なくとも最初の10〜14日）は**「回復と再構築」**を最優先とします。後半から徐々にクオリティを整理した形で再導入します。

**7月の数値目標：総距離160〜175km（6月より微減または同程度）・クオリティセッション週2回厳守・R走は1セッションを上限**

### 特に意識してほしいこと（7月の行動指針）

**① 1日1アクティビティを原則とする**
ウォームアップ・メイン・クールダウンを別々に記録するのをやめ、1セッションとして完結させてください。これにより負荷の全体量が正確に把握でき、Garminの判定精度も上がります。

**② ロング走中のペースアップは禁止**
ロング走はE強度（5:40〜6:10/km）で最初から最後まで一定に走ること。「気持ちよくなってきたからペースを上げる」という行動が、今月の過剰負荷の主因の一つです。

**③ 高強度翌日はE走か完全休養**
R走・T走・ロング走の翌日は、必ずE走（8km以内）か休養にする。これだけで疲労の蓄積ペースが大幅に改善します。

**④ 睡眠6.6時間→7時間以上へ**
ダニエルズは練習だけでなく回復の質を非常に重視します。睡眠の0.4時間の差が、月30日でHRV・レディネス・VO2max推定に明確に出てきます。「練習を1日増やす」よりも「睡眠を30分増やす」を7月の最重要施策として取り組んでください。
