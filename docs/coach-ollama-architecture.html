coach_ollama.py 内部処理

処理ステップ

1. CSV 読込 走行・ラップデータを Markdown 表に変換。週別距離集計。
2. プロンプト構築 ダニエルズ VDOT / 5ゾーン / 80-20 ルールを system prompt に埋込。
3. Ollama API POST /api/chat ストリーミング。月間総評・週別・翌月提案。
実行コマンド
# データ取得 python3 strava_fetch.py # Ollama.app 起動後 python3 coach_ollama.py python3 coach_ollama.py --month 2026-06 # 環境変数（任意） OLLAMA_HOST=http://127.0.0.1:11434 OLLAMA_MODEL=gemma4:12b python3 coach_ollama.py
推奨運用（夜間バッチ）

名刺アプリと同じパターン

日常 update_report.sh で HTML レポート更新（GitHub Pages）
月1回 cron で coach_ollama.py --month → MD レポート生成
インフラ 自宅 Mac mini M2 + Ollama.app（ollama-app 0.30.6）
技術メモ
vision 不要
ランニングコーチは CSV テキストのみ。名刺/レシートより実装が軽い。gemma4 の thinking で 1〜3 分。
依存関係
coach_ollama.py は標準ライブラリのみ。anthropic パッケージ不要。Ollama 起動が前提。
