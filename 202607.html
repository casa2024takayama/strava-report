2026年7月 ランニングレビュー（7月5日時点・中間評価）

1. 月間総評

現時点の走行距離：28.5km（5日経過・進捗16%）

進捗16%の時点で28.5kmを記録しています。同ペースで推移した場合、月末には約178km見込み（目標200kmに対して約89%）となります。現時点での距離は経過日数に対して概ねペース通りですが、若干のビハインドがあるため、残り26日でコンスタントに走る必要があります。

ただし、距離よりも体の状態が大きな懸念点です。Garminの生理指標を見ると、トレーニングステータスが「STRAINED（ひずみ状態）」、トレーニングレディネスが平均20/100（範囲1〜44）、HRVがUNBALANCED、睡眠スコア59点・睡眠時間5.5時間と、疲労と回復不足の警戒シグナルが複数点灯しています。安静時心拍の50→51bpmへの上昇も蓄積疲労のサインです。

量の確保を焦る前に、まず回復を優先することが7月後半の最優先課題です。

2. 週別評価

第1週（7/1〜7/5）：28.5km／3回

良い点：

  • 3回の練習でそれぞれ異なる刺激（ゆっくりジョグ、ドリル系、ビルドアップ的な高強度走）を実施しており、単調ではない練習構成。
  • 7/2のNight Runは序盤の心拍が120〜134bpmと低く抑えられており、イージーゾーンでの走行を意識した形跡が見られる。

課題：

  • 7/5に2本（Evening RunとNight Run）を同日実施しており、特にEvening Runが非常に高強度だったことを考えると、同日2部練の負荷は過大。
  • トレーニングレディネスが平均20という極めて低い値にもかかわらず高強度走を行っており、回復が追いついていない状態での練習強行が見受けられる。
  • 睡眠5.5時間は回復には明らかに不足しており、練習の質・量よりも睡眠改善が急務。

3. 個別アクティビティコメント

✅ 2026-07-02（木）Night Run 10.3km・5:48/km・HR136

良い点：

  • ラップ1〜5（6:07〜5:46/km、HR120〜134）はイージーゾーンで良好な有酸素刺激。VDOT57のEペースは概ね5:50〜6:10/km相当であり、ほぼ適正範囲。
  • 後半のドリル系（ラップ8〜17：短い全力疾走＋歩きまたはジョグ）はR（レペティション）ゾーン相当の刺激として機能している。短距離での3:21〜3:59/kmは走経済性への良い刺激。

改善点：

  • ラップ6（4:49/km・HR163）がドリル前に突然混入しており、ペース変動の意図が不明確。IゾーンとEゾーンの切り替えが唐突。ウォームアップ→ドリル→クールダウンという明確な構造にするとより効果的。

⚠️ 2026-07-05（日）Evening Run 14.0km・4:44/km・HR159

良い点：

  • ラップ1〜3（5:53〜5:32/km、HR127〜136）はイージーゾーンで丁寧なウォームアップ。
  • ラップ7〜14（4:18〜4:02/km、HR171〜188）は明らかにI（インターバル）ゾーン以上の高強度。VDOT57のIペース目安は約4:00〜4:10/km。スピード自体は走力に見合っている。

改善点：

  • 問題の核心は「いつ・どういう状態で走ったか」。トレーニングレディネスが極めて低い状態で、最大HR198bpm（最大心拍数の上限に近い）まで達するような高強度走を行っており、オーバーリーチの典型パターン。
  • ラップ5（5:19/km）→ラップ6（4:43/km）→ラップ7（4:18/km）と急激にペースアップしており、ウォームアップが終わりきらないうちに高強度に突入している。
  • 後半のラップ11〜12でやや落ちた後、ラップ13〜14でさらに加速しており、疲労した状態での無理な追い込みが見られる。HRmax 198はこの状況での上限値として心臓への過負荷リスクがある。

⚠️ 2026-07-05（日）Night Run 4.2km・6:14/km・HR153

良い点：

  • ペース自体はEゾーン相当で、クールダウン的な意図は理解できる。

改善点：

  • Evening Runの直後（同日）に実施しており、インターバル走直後の2部練は回復の観点から非推奨。
  • ラップ3〜10（0.06〜0.11kmの細かいラップ・3:06〜9:43/kmの激しいペース変動）が含まれており、ゆっくり回復する意図とは矛盾した動きが散在している。この短い全力疾走は既に疲弊した筋肉への余分なストレスとなる。
  • このランはクールダウンウォークに留めるべき状況だった。

心拍の観点：

  • Evening RunのHRmax 198bpmは、VDOT57のランナーとして心肺への相当な負荷を示している。最大心拍数が200前後と仮定するとHRmax近傍での持続走となり、レディネスが低い状態では回復に3〜5日必要なレベルの刺激。
  • 7/2 Night RunはHR120〜136帯（イージー）で序盤を過ごせており、有酸素基礎の構築には良い心拍管理。

負荷バランスの問題：

  • Garminが「ANAEROBIC_SHORTAGE（無酸素系の刺激不足）」と判定しているが、7/5 Evening RunのようなI〜最大強度走を連日行うことで解決しようとするのは誤り。7/2にもドリル系のR刺激を入れており、1週間の中で無酸素系の刺激は十分。GarminのANAEROBIC_SHORTAGEは状態の評価より機械的な積算のズレが出ている可能性があり、現状のSTRAINEDとの矛盾を考えると信号を鵜呑みにせず体の感覚を優先すべき。

5. 2026年8月の練習提案

前提・設計方針

Garmin生理指標から読み取る重要事項：

  • STRAINEDが月内4回、レディネス平均20、HRV UNBALANCED、睡眠5.5h、安静時HR上昇——これらはすべてオーバーリーチの複合的な警告。
  • 8月は「まず回復してからベースを積む」月とする。最初の1週間は強度を落とし、体を立て直してから段階的に質と量を上げる構成にする。
  • 8月のレースなし。月間200km・週平均40km（5週想定）を目標とするが、第1週は回復週として30kmに抑え、後半週で補う。
  • ダニエルズの80/20原則に従い、クオリティセッション（T/I/R）は週2回上限、残りはE走。
  • 睡眠改善（目標7時間以上）が練習効果を最大化する最重要施策であることを強調する。
